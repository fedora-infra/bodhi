======================
bodhi-monitor-composes
======================

Synopsis
========

``bodhi-monitor-composes``


Description
===========

``bodhi-monitor-composes`` prints a human-readable report about the running Composes.


Options
=======

``--help``

    Display help text.

``--version``

    Report the Bodhi version and exit.
