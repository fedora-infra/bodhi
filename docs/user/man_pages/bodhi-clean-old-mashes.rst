======================
bodhi-clean-old-mashes
======================

Synopsis
========

``bodhi-clean-old-mashes``


Description
===========

``bodhi-clean-old-mashes`` deletes all but the 10 newest mash directories.


Options
=======

``--help``

    Display help text.

``--version``

    Report the Bodhi version and exit.
