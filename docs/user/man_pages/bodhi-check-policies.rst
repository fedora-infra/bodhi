====================
bodhi-check-policies
====================

Synopsis
========

``bodhi-check-policies``


Description
===========

``bodhi-check-policies`` iterates over Updates to check the policies enforced by
Greenwave for each update.


Options
=======

``--help``

    Display help text.

``--version``

    Report the Bodhi version and exit.
