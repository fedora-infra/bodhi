===========
bodhi-shell
===========

Synopsis
========

``bodhi-shell``


Description
===========

``bodhi-shell`` is used to run a Python shell initialized with Bodhi models.


Options
=======

``--help``

    Show help text and exit.

``--version``

    Show version and exit.
